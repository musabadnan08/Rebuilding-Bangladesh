Frequently Asked Questions.

What is the Document Object Model (DOM)?

Document Object Model (DOM) is a interface for web documents. DOM represents the structure of the webpage as a tree of objects. Every HTML element is considered as a node. By using JavaSript these nodes can be accessed, modified, added or removed and this is a way to change the content or behavior of a webpage and it can make the webpage functional.

How do you select an element from the DOM?

There are five methods for selecting elements from the DOM. getElementById -> This can be used to select a single element by its unique ID. querySelector -> This can be used to select the first element matching a CSS selector. querySelectorAll -> This can be used to select all elements matching a CSS selector. getElementsByClassName -> This can be used to select elements by their class name. getElementsByTagName -> This can be used to select elements by their tag name.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a way, where an event listener can be added to a parent element instead of each child. When a event happens on a child element, it bubbles up to the parent. The parent element can check which child element triggered the event. This is a useful way because if we use this way we don't need to add multiple event listeners for each child and it also works for dynamically added child elements.

How do you manipulate an element's attribute and styles using the DOM?

For manipulating an elements attributes in the DOM. we can use methods like setAttribute() to add or change and attribute, getAttribute() to read it, and removeAttribute() to delete it. To change styles, we can use the style properties to directly modify single css properties. For multiple changes, we can use cssText. And there is a another way to manage styles by adding, removing or toggling between classes using classList.
